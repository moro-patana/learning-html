List of attribute rules

Attribute rules are scripts that a user can define on datasets. They trigger automatically with incoming edits and can be used to constrain the attribute values allowed on fields or perform a calculation to derive a field's values.

Do not do this:

  • ......<section id= section_jpg >
    ........<header>
    ..........<h2>As a JPEG</h2>
    ............<p>Joint Photographic Experts Group</p>
    ........</header>
    ............<img src=img/onja_crew.jpg alt=students>
  • ......<section id="SECTION_jpg">
    ........<header>
    ..........<h2>As a JPEG</h2>
    ............<p>Joint Photographic Experts Group</p>
    ........</header>
    ............<img src="IMG/ONJA_CREW.jpg" alt="STUDENTS">

Do this instead:

Attribute values must have quotes. Attribute names need to be lowercase.

  • ......<section id="section_jpg">
    ........<header>
    ..........<h2>As a JPEG</h2>
    ............<p>Joint Photographic Experts Group</p>
    ........</header>
    ............<img src="img/onja_crew.jpg" alt="students">
  • ......<section id="section_jpg">
    ........<header>
    ..........<h2>As a JPEG</h2>
    ............<p>Joint Photographic Experts Group</p>
    ........</header>
    ............<img src="img/onja_crew.jpg" alt="students">
