Indenting

  • This is a great visual benefit for the author during code production, and for others that may view the source code afterward.
  • You can not close the parents tags if you did not close the child tags yet.

Void elements

  • The void elements are ones that do not have a closing tag and therefore cannot have content placed inside them, and they make no nest.
    In HTML, using a closing tag on an empty element is usually invalid. For example, <input> type=text></input>.
    The void elements are: br, col, hr, img, input, link, meta, source, track.

Elements

  • The first and most important thing to understand in HTML syntax is the usage of elements and tags. Every HTML element is defined by an opening tag and a closing tag wrapped around their content.
