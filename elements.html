HTML elements

Definition

An HTML element element is an individual component of an HTML document. It represents semantics, or meaning. For example, the title element represents the title of the document. Most HTML elements are written with a start tag (or opening tag) and an end tag (or closing tag), with content in between. Elements can also contain attributes that defines its additional properties.
