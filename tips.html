Tips on how to write HTML

How to write HTML quicker and what do we use?

  • Use keyboard shortcut:
    Select the word which you want to repeat. Press Shift+right arrow. Then press ctrl+D until all the words are selected and move the cursor at the beginning of the word which you selected. Now you can write what you want.
  • Use a code editor that has auto-completion. Auto-completion allows you to start typing a tag or attribute and the software predicts what youâ€™re typing and finishes it for you. This means you can type just a couple of characters and the return key to type considerably longer bits of code.
  • Use Emmet
    You write simple abbreviations and just press Tab or CTRL+E or any other other supported keyboard shortcut and Emmet expands the simple abbreviations into complex HTML and CSS code snippets. Emmet will make your HTML and CSS workflow much faster.

How to make sure your work is high quality?

  • Reread what you have written and if it is ok for you, test it in HTML5 validator. Correct the mistakes and copy the correct version to your code. Finally, make sure your indenting is easy to read and understandable.
